# uddental

치과 웹사이트 구축 + 광고 전략/레퍼런스 수집 프로젝트

## 폴더 구조

- `site/` : 치과 사이트 개발
- `ads-research/` : 광고 관련 자료 정리
  - `competitors/` : 경쟁사 광고/랜딩 분석
  - `keywords/` : 검색 키워드 리서치
  - `creatives/` : 광고 카피/이미지/영상 아이디어
  - `landing-pages/` : 랜딩페이지 벤치마크
  - `notes/` : 전략 메모

## 시작 아이디어

1. 사이트 MVP 구성 (홈, 진료과목, 의료진, 상담예약)
2. 광고 채널 우선순위 결정 (네이버 검색/플레이스, 인스타, 유튜브 등)
3. 월 예산 가정별 시나리오 작성 (저/중/고)
4. KPI 설정 (상담신청 수, CPA, 전환율)
