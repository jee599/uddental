# 자동화 범위 + 우리가 더 나을 수 있는 전략 (병원/치과 네이버 광고)

작성일: 2026-03-12

## 1) 우리가 자동화로 할 수 있는 범위 (현실 수치)

### High confidence 자동화 (70~90%)
1. 키워드 분류/군집화 (브랜드/일반/지역/경쟁)
2. 제외키워드 추천 (검색어 리포트 기반)
3. 소재 성과 모니터링/중지 후보 추천
4. 일간 이상치 탐지(CPC 급등, CVR 급락)
5. 주간/월간 리포트 자동 생성
6. 플레이스 운영지표 점수화(리뷰 최신성/응답률/정보완성)

### Medium 자동화 (50~70%)
1. 예산 재배치 추천(CPA 기준)
2. 광고문안 초안 생성 + 위험표현 필터링
3. 랜딩 개선안 제안(체크리스트 기반)
4. 리마케팅 세그먼트 추천

### Low 자동화 (20~40%)
1. 의료광고 심의 이슈 최종 판정
2. 원장/실장 커뮤니케이션 및 클레임 대응
3. 경쟁 급변 시 전략 피벗 결정

결론: 총 자동화 가능 범위는 **55~70%**

## 2) 우리가 기존 업체보다 더 나아질 수 있는 지점

## 우위 A: 운영 빈도
- 기존: 주간 최적화
- 우리: 일간 감시 + 즉시 알림 + 자동 제안

## 우위 B: 통합 운영
- 기존: 광고/플레이스/콘텐츠 분절
- 우리: 3채널 KPI를 동일 키워드 체계로 통합

## 우위 C: 설명 가능한 운영
- 기존: “최적화했습니다” 보고
- 우리: "어떤 신호로 어떤 조치를 했는지" 로그 제공

## 우위 D: 의료 특화 가드레일
- 위험 문구 자동 감지
- 과장/단정형 표현 사전 경고

## 3) 우리가 실제 쓸 전략 세트 (실행형)

### 전략 1: 고의도 키워드 압축
- 예산 70%를 고의도 그룹(지역+시술+가격/후기)에 고정
- 저의도 broad는 테스트 한도 설정

### 전략 2: 손실 차단 루프
- 3일 연속 고비용 무전환 키워드 자동 경고
- 7일 기준 제외키워드 추천 + 소재 교체 제안

### 전략 3: 플레이스 품질 루프
- 리뷰 최신성/응답률 임계치 미달 시 운영 태스크 자동 생성
- 사진/소식 업데이트 누락 경고

### 전략 4: 콘텐츠 환류 루프
- 광고 고비용 키워드를 블로그 SEO 주제로 전환
- 동일 키워드군에서 유료/유기 성과 동시 모니터링

### 전략 5: 원장 1페이지 리포트
- KPI 5개만 고정: CPA, CPL, 콜연결률, 예약전환율, 손실키워드비중
- 매주 같은 포맷으로 제공

## 4) 운영 로드맵

- Week 1~2: 데이터 연결 + 기준선 측정
- Week 3~4: 자동 추천 엔진 v1 적용
- Week 5~6: 플레이스/콘텐츠 연동
- Week 7~8: 파일럿 전후 성과 리포트

## 5) 리스크와 대응

- 리스크: 심의/법적 표현 이슈
  - 대응: 사전 금칙어 + 사람 검수
- 리스크: 전환 데이터 누락
  - 대응: 콜트래킹/예약 이벤트 점검 체크리스트
- 리스크: 자동화 과신
  - 대응: 주 1회 운영자 승인 워크플로
