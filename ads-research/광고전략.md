# 동백유디치과 광고 전략 (2026년 3월) — v2

작성일: 2026-03-20

근거: 네이버 SA/플레이스/블로그 알고리즘 리서치, 치과 광고업체 벤치마크, 동백 지역 경쟁 분석

## 핵심 요약

동백유디치과의 최대 무기는 **임플란트 30~60만원**(업계 평균 130만원 대비 절반 이하)이다. 동백 지역 치과 14개+ 중 대부분이 온라인 리뷰 0건, 플레이스 미최적화 상태로 **지금이 디지털 마케팅 선점의 골든타임**이다.

네이버 3축(검색광고 + 플레이스 + 블로그)을 중심으로 카카오/구글/오프라인까지 5채널 통합 운영이 핵심이다.

## Part 1. 시장 환경

### 1.1 동백 지역 현황

| 항목 | 수치 |
|------|------|
| 동백동 인구 | 약 86,000명 (동백1/2/3동 합산) |
| 유소년(15세 미만) 비율 | 21.4% (용인시 평균 16.1% 대비 높음) |
| 지역 내 치과 수 | 14개+ |
| 치과당 인구 | 약 6,100명 |
| 핵심 인구 특성 | 자녀를 둔 30~40대 가족 중심 |
| 주요 아파트 단지 | 호수마을, 백현마을, 성산마을, 어은목마을, 초당마을 + 신동백 3,500세대+ |
| 학교 | 초등 6곳, 중등 4곳, 고등 3곳 |

**지역 특성:** 4면이 산으로 둘러싸여 동백 내부에서 소비가 집중되는 구조. 출퇴근 시간 도로정체 심각 → "동네 치과" 수요 높음. 지역 밀착 마케팅에 최적화된 환경.

### 1.2 경쟁 구도

**경쟁사 Top 3:**

| 치과 | 거리 | 강점 | 약점 | 온라인 리뷰 |
|------|------|------|------|-----------|
| **로덴치과** | 50m | 서울대 박사, "치과주치의" 브랜딩 | 목요일 휴진, 높은 가격대 | 확인 필요 |
| **동백센트럴치과** | 500m | 11개 진료과, 야간진료(화/목 20:30) | 온라인 평판 부재 | 굿닥 0건 |
| **동백탑치과** | 1km | 모두닥 평점 4.6 (지역 최고) | 중심 상권에서 떨어진 위치 | 7건 |

**기회 요인:** 대부분 경쟁 치과가 네이버 리뷰 0건. 리뷰 20건만 확보해도 압도적 우위.

### 1.3 환자가 치과를 고르는 실제 기준 (우선순위)

1. **거리/접근성** — 도보 10분 이내, 출퇴근 동선 (동백에서는 압도적 1순위)
2. **지인 추천** — 가족, 이웃, 맘카페 추천 (특히 아파트 단지 커뮤니티)
3. **가격** — 임플란트, 교정 같은 고단가 시술에서 결정적
4. **온라인 리뷰/분위기** — 플레이스 사진, 리뷰 내용
5. **원장 경력/전문성**

→ 온라인 광고는 3~5번에 영향을 주지만, 1~2번을 간과하면 전환이 안 된다.

### 1.4 치과 광고 시장

- 전국 치과의원 약 19,000개. 서울은 이미 폐업 > 신규 개원
- 치과 매출 감소 요인 1위: 경쟁 과열로 인한 환자 수 감소 (77.5%)
- 월 마케팅 비용 평균: 100~600만원. 적정 예산은 총 매출의 10%
- 네이버 치과 키워드 CPC: 1~4만원 (지역 키워드는 2,000~5,000원)

## Part 2. 네이버 3축 알고리즘 분석

### 2.1 네이버 검색광고 (SA)

**순위지수 = 입찰가 × 품질지수**

품질지수 구성 요소 (추정 가중치):

| 요소 | 추정 비중 |
|------|----------|
| 클릭률(CTR) | 40~50% |
| 키워드-소재 연관성 | 20~25% |
| 키워드-사이트 연관성 | 20~25% |
| 사용자 경험 데이터 (체류시간, 이탈률) | 10~15% |

**2025-2026 변화:** LLM 도입으로 사용자 행동 데이터(체류시간, 스크롤 깊이, 이탈률) 비중 급상승.

**품질지수 7칸 달성 핵심:**
1. 키워드와 카테고리 정확히 일치
2. 랜딩페이지에 키워드 자연스럽게 삽입
3. 확장소재 10종 전부 세팅 (전화번호, 위치, 이미지, 서브링크, 네이버예약)
4. 제외키워드 그룹당 70개 + 소재당 70개 = 최대 140개
5. 소재에 대체키워드 삽입
6. MO/PC 분리 운영

### 2.2 네이버 플레이스

**2025년 5월 대폭 변경: 리뷰 수 중심 → 실제 사용자 행동 데이터 중심**

랭킹 요소 (추정 우선순위):

| 순위 | 요소 | 설명 |
|------|------|------|
| 1 | 사용자 행동 데이터 | 클릭, 전화, 길찾기, 예약 시도, 체류시간 |
| 2 | 자연스러운 유입 비율 | 비정상 패턴 감지 시 노출 제한 |
| 3 | 재방문/순환 구조 | 검색→클릭→재방문 반복 |
| 4 | 정보 완성도 | 80% 이하면 노출 현저히 하락 |
| 5 | 리뷰 최신성 | 총 리뷰 수보다 **최근 1개월 리뷰 수**가 핵심 |
| 6 | 네이버 솔루션 활용도 | 예약(3.1배 유입 증가), 톡톡, 스마트콜 |
| 7 | 블로그/콘텐츠 연동 | 블로그 리뷰, 소식 발행 빈도 |

**2026년 리뷰 정책 강화 (1월 22일~):**
- AI 기반 리뷰 어뷰징 단속 대폭 강화. 1회 적발만으로 업체 단위 페널티 즉각 적용
- 리뷰 탭 블라인드 + AI 브리핑 메뉴 전체 삭제. 마케팅 대행사도 단속 대상
- "리뷰 쓰면 OO 드립니다" 방식은 어뷰징으로 적발될 수 있음 → 자연스러운 유도만 가능

### 2.3 네이버 블로그 (C-Rank / D.I.A)

**C-Rank:** 블로그 자체의 신뢰도/전문성 평가
- Context(주제 일관성) × Content(정보 품질) × Chain(사용자 반응) × Creator(운영 이력)

**D.I.A:** 개별 문서의 검색 의도 충족도 평가
- 딥러닝 기반 의도 분석, 문서 패턴 인식, 동적 랭킹

**2025-2026 변화:**
- AI 콘텐츠 탐지/제재 강화
- VIEW 탭 → 스마트블록 통합
- 네이버 AI 검색(큐:)에서 인용되려면 정의형 문장, H2/H3 계층화, FAQ 포함 필수
- 체류시간 권장: 130~150초 (약 2분 30초~3분)

**저품질 판정 기준:** 키워드 도배, 복제 콘텐츠, 유사 글 반복, 과도한 상업성, AI 생성 콘텐츠. 의료 분야는 저품질 판정 기준이 더 엄격.

## Part 3. 의료광고 규정

### 3.1 사전심의

- 일평균 10만 명 이상 이용 플랫폼(네이버, 카카오, 유튜브 등)의 의료광고는 사전심의 필수
- 심의 기관: 대한치과협회 의료광고심의위원회 (접수 후 최소 7~10영업일 소요)
- 병원 자체 홈페이지(자체 도메인)는 예외 가능

### 3.2 금지 표현

| 유형 | 예시 |
|------|------|
| 절대적 효과 보장 | "100% 안전", "영구적 효과 보장" |
| 비교 우위 | "국내 최고", "업계 1위" |
| 환자 치료경험담 | 대가성 후기 위장 |
| 비급여 할인 광고 | 할인 금액/대상/기간 구체 명시 |
| 시술 연상 이미지 | 임플란트 모형, 의료기기 이미지 |
| 최상급 표현 | "최고", "1등", "유일한" |
| **"반값 임플란트"** | **할인으로 해석될 수 있어 심의 위반 리스크 높음** |

### 3.3 블로그 운영 시 의료법 주의

- **정보성 글은 치과명 없이 순수 정보로** 작성. 치과 소개가 포함되면 의료광고로 분류
- Before/After 사진은 개인정보보호법 + 의료광고법 이중 규제. 2025년 이후 적발 급증
- 플레이스 리뷰 답변에도 의료광고법 적용. "다음에 오시면 할인해드리겠습니다" = 환자 유인
- 가격 표기 시 "정상가"임을 명확히 하고, 재료/등급별 가격 차이 구체적 명시 필요

### 3.4 처벌

- 거짓/과장 광고: 1년 이하 징역 또는 1,000만 원 이하 벌금
- 환자 유인/알선: 3년 이하 징역 또는 3,000만 원 이하 벌금
- 2024년 불법 의료광고 366건 적발 (자발적 후기 위장 31.7%, 비급여 할인 26.7%)

## Part 4. 키워드 전략

### 4.1 키워드 포트폴리오

| 유형 | 비중 | 예시 | CPC 추정 |
|------|------|------|---------|
| 핵심 키워드 | 40% | 임플란트, 치아교정 | 2~5만원 |
| 지역 키워드 | 30% | 동백 치과, 동백 임플란트, 동백역 치과 | 2,000~5,000원 |
| 롱테일 키워드 | 30% | 동백 임플란트 가격, 동백 야간 치과 | 1,000~3,000원 |

### 4.2 우선순위 키워드

**Tier 1 (필수 확보):**
- "동백 치과" — 최고 전환 가능성
- "동백 임플란트" — 고단가 시술 + 가격 강점
- "동백역 치과" — 역세권 유입

**Tier 2 (확장):**
- "동백 치과 추천" — 의사결정 단계
- "동백 교정", "동백 소아치과" — 가족 단위
- "어정역 치과" — 인근 역 커버

**Tier 3 (롱테일):**
- "동백 임플란트 가격" — 가격 강점 부각
- "용인 임플란트 가격", "기흥구 임플란트" — 넓은 지역
- "동백 스케일링", "동백 사랑니" — 진입 시술

### 4.3 블로그 콘텐츠 키워드

| 콘텐츠 유형 | 비율 | 의료법 리스크 |
|------------|------|-------------|
| 질환 정보성 (충치 원인, 잇몸병 예방) | 50~60% | 낮음 |
| 치료 과정 안내 (임플란트 과정, 교정 종류) | 20~30% | 중간 (심의 필요) |
| 구강건강 팁 (칫솔질, 식습관) | 10~15% | 낮음 |
| 원장/의료진 소개 | 5% | 중간 |

## Part 5. 실행 전략

### 실행 전 필수 확인 사항 (프랜차이즈 제약)

유디치과 프랜차이즈 소속으로 아래 사항을 본사에 먼저 확인해야 한다. 확인 안 되면 전략의 절반이 실행 불가할 수 있다.

- [ ] 개별 점포 네이버 블로그/SNS 운영 가능 여부
- [ ] 플레이스 정보 수정 권한 (개별 점포에 있는지, 본사 통합 관리인지)
- [ ] 독자 검색광고 집행 가능 여부
- [ ] 가격 광고 범위 (본사 가격 정책 밖의 프로모션 가능 여부)
- [ ] "동백유디치과" 로컬 브랜딩 허용 여부
- [ ] 랜딩페이지: 동백점 개별 홈페이지가 불가하면 네이버 플레이스 예약 페이지를 랜딩으로 사용

### Week 1 퀵윈 (비용 0원, 당일 실행)

| 작업 | 소요시간 | 비용 |
|------|---------|------|
| 플레이스 정보 100% 채우기 | 2시간 | 0원 |
| 구글 비즈니스 프로필 등록 | 1시간 | 0원 |
| 카카오톡 채널 개설 | 30분 | 0원 |
| 기존 내원 환자 10명에게 리뷰 요청 (구두/문자) | 1시간 | 0원 |
| "동백 치과" "동백 임플란트" SA 2개 키워드, 일예산 2만원 시작 | 1시간 | 일 2만원 |

→ 이 5가지를 첫 주에 끝내면, 2주차부터 유입 데이터가 쌓인다.

### Phase 1: 기반 구축 + SA 시작 (1~3주)

**플레이스 최적화:**
- [ ] 정보 완성도 100% 달성 (진료시간, 주차, 편의시설, 진료항목+가격)
- [ ] 고품질 사진 20장 이상 등록 (외관, 진료실, 대기실, 장비, 스태프)
- [ ] 소개글 200자 이상 (첫 문장에 "동백", "기흥" 지역 키워드)
- [ ] 해시태그: #동백치과 #동백임플란트 #동백역치과 #용인임플란트 #동백스케일링
- [ ] 네이버 예약 연동 (유입 3.1배 증가 효과)
- [ ] 스마트콜 연동 (유입 1.2배 증가 + 전화 추적)
- [ ] 톡톡 연동

**멀티채널 세팅:**
- [ ] 구글 비즈니스 프로필 등록 및 최적화 (사진, 진료시간, 카테고리)
- [ ] 카카오톡 채널 개설 + 진료예약 리마인드 자동 메시지 세팅
- [ ] 인스타그램 비즈니스 계정 개설

**광고 기초:**
- [ ] 기존 의료광고 콘텐츠 전수 점검 (심의 미통과 콘텐츠 삭제/수정)
- [ ] 대한치과협회 심의 인증번호 신청 (7~10영업일 소요)
- [ ] 네이버 SA 지역 키워드 2~3개로 즉시 시작 (일예산 2~3만원)
- [ ] UTM 파라미터 설계 (채널별 유입 추적)

**전화 응대 프로세스 구축:**
- [ ] 전화 응대 스크립트 작성 (가격 문의 대응, 상담 예약 유도, 불안 해소 멘트)
- [ ] 점심시간(12~13시) 전화 응대 인력 확보 (이 시간대 SA 입찰 상향 시 필수)
- [ ] 첫 방문 접수지에 "치과를 어떻게 알게 되셨나요?" 항목 추가 (유입 경로 추적)

### Phase 2: 리뷰 확보 + 블로그 + 지역 밀착 (2~6주)

**리뷰 캠페인:**
- [ ] 수납 시 태블릿으로 바로 리뷰 작성 유도 (QR코드는 집에 가면 안 한다)
- [ ] 목표: 첫 3개월 누적 20건 (월 5~8건이 현실적)
- [ ] 모든 리뷰 24시간 내 답변 (긍정/부정 불문, 의료광고법 주의)
- [ ] 영수증 리뷰 자연 유도 (신뢰도 최고, 순위 영향 최대)
- [ ] 구글 리뷰도 병행 수집 (규제가 상대적으로 느슨)

**블로그 운영:**
- [ ] 동백유디치과 네이버 블로그 개설 (본사 승인 후)
- [ ] 주 1~2회 정보성 포스팅 (1,500~3,000자, 이미지 5~10장)
- [ ] 워크플로: 원장이 핵심 팩트 5분 음성 녹음 → AI 초안 → 원장 팩트 체크
- [ ] 정보성 글은 치과명 없이 순수 정보로 작성 (의료법 분리)
- [ ] 치과/구강건강 전문 블로그로 주제 집중 (C-Rank 축적)

**지역 밀착 마케팅 (오프라인 연계):**
- [ ] 동백 맘카페(네이버 카페) 활동 — "동백 치과 추천해주세요" 글에 자연 언급 유도
- [ ] 초등학교 구강검진 무료 봉사 → 소아치과 환자 유입의 가장 강력한 채널
- [ ] 동백 주요 아파트 엘리베이터 광고 검토 (월 50~100만원)

**플레이스 소식:**
- [ ] 주 2회 소식 발행 (진료 안내, 의료 정보, 비하인드)
- [ ] 사진 매주 1~2장 추가 업로드

### Phase 3: 광고 확장 + 최적화 (1~3개월)

**네이버 SA 확장:**
- [ ] 일예산 3~5만원으로 확대 (월 최소 100만원)
- [ ] 캠페인 분리: 브랜드 / 지역+시술 / 롱테일
- [ ] 확장소재 10종 전부 세팅 (전화번호, 위치, 이미지, 서브링크, 네이버예약)
- [ ] 2~4주 균등 운영 → 데이터 기반 시간대/요일별 입찰 조정
- [ ] 진료시간(09:30~18:30) 입찰 강화, 토요일 오전 별도 입찰, 일요일 OFF
- [ ] PC/MO 분리 운영
- [ ] 제외키워드 설정 (무료, 채용, 후기알바, 자격증 등)

**플레이스 광고:**
- [ ] 일예산 5만원으로 테스트
- [ ] "동백 치과", "동백 임플란트" 핵심 키워드 집중

**A/B 테스트:**
- [ ] 광고 소재: 가격 강조 vs 신뢰 강조("정품 재료, 정직한 진료") vs 긴급성
- [ ] CTA: "전화 상담" vs "네이버 예약" vs "카톡 상담"
- [ ] 소재당 최소 100~200클릭 필요 → 1라운드에 2~4주 소요
- [ ] 승패 판정: CTR + 전환율(전화/예약) 동시 평가

### Phase 4: 고도화 + 리마케팅 (3~6개월)

**리마케팅 전략:**
- [ ] 네이버 GFA 리타게팅: 랜딩페이지 방문 후 이탈자에게 배너 노출
- [ ] 리마케팅 세그먼트: 임플란트 페이지 조회자 (고가 시술, 재노출 가치 높음)
- [ ] 빈도 제한: 일 3회 이하 노출 캡

**광고 최적화:**
- [ ] 3일 연속 무전환 키워드 자동 경고
- [ ] 7일 누적 손실 키워드 후보화
- [ ] 자동입찰 도구 검토 (오토다비드, 보라웨어 등)
- [ ] GFA 성과형 디스플레이 테스트 (지역+관심사 타겟팅)
- [ ] 카카오 비즈보드 광고 테스트 (동백동 반경 3km 타겟, CPC가 네이버 대비 30~50% 저렴)

**인스타그램/숏폼:**
- [ ] 주 2회 릴스 발행 (블로그 콘텐츠 재가공)
- [ ] 콘텐츠 방향: 원장님의 구강건강 팁 60초, 스케일링 과정 타임랩스

**환자 리텐션(재내원) 전략:**
- [ ] 카카오톡 채널 자동 메시지: 정기검진 리마인드(6개월), 스케일링 알림(1년)
- [ ] 진료 후 만족도 문자 → 만족 시 리뷰 유도, 불만족 시 내부 피드백
- [ ] 가족 유입: 1명 내원 시 가족 검진 안내 (동백 가족 중심 지역 특성 활용)

**채널 다변화 목표:**
- 트래픽 소스 비율: 네이버 60% / 구글 15% / 카카오 15% / 인스타/SNS 10%
- 네이버 의존도 50% 이하로 낮추기 (알고리즘 변경 리스크 헤지)

- **접근성**: 동백중앙로 핵심 상권, 주차 가능
- **로컬 정체성**: "동백유디치과"로 독립적 브랜딩 (본사 허용 시)

## Part 8. 전환 추적 및 데이터 설계

### 8.1 전환 추적 체계

| 전환 유형 | 추적 방법 |
|----------|----------|
| 전화 | 스마트콜(050번호) + 캠페인별 별도 추적번호 |
| 네이버 예약 | 네이버 예약 API (유입 경로별 분리 집계) |
| 카카오 상담 | 카카오톡 채널 유입 경로 추적 |
| 워크인 | 접수지 "어떻게 알게 되셨나요?" 항목 |
| 블로그 유입 | UTM 파라미터 (`utm_source/medium/campaign/content`) |

### 8.2 환자 퍼널

```
검색/노출 → 클릭 → 전화/예약 → 내원 → 시술 → 재방문/추천
             ↓        ↓           ↓       ↓
           CTR    전환율 3~5%  내원율 50~70%  LTV
```

각 단계별 전환율을 추적하여 병목 구간을 식별한다.

### 8.3 채널별 기여도

Last-click 기준으로 시작. SA에서 블로그를 보고 플레이스에서 전화한 환자 → 플레이스에 귀속. 추후 데이터가 쌓이면 멀티터치 모델 검토.

### 8.4 데이터 파이프라인

```
[소스]                         [통합]              [의사결정]
네이버 SA 리포트 ──┐
플레이스 통계 ──────┤
스마트콜 로그 ──────┼──→ 구글 스프레드시트 ──→ 주간 리포트
블로그 애널리틱스 ──┤     (자동 수집 가능 항목)     의사결정 트리거
네이버 예약 ────────┤
접수지 유입 경로 ───┘     (수동 입력)
```

**의사결정 트리거:**
- CPA가 10만원 초과 → 해당 키워드 입찰가 조정 또는 OFF
- 3일 연속 무전환 키워드 → 경고
- 품질지수 4칸 이하 → 소재 교체 검토

## Part 10. LTV(환자 생애가치) 기반 전략

### 시술별 LTV 모델

| 시술 | 1회 매출 | 추정 LTV | 허용 CPA |
|------|---------|---------|---------|
| 스케일링 | 5만원 | 연 2회 × 5년 = 50만원 | 5만원 |
| 충치/보존 | 10~30만원 | 재방문 포함 100만원 | 10만원 |
| 임플란트 | 30~60만원 | 사후관리 포함 100~200만원 | 20만원 |
| 교정 | 200~400만원 | 300~500만원 | 30만원 |
| 소아 (정기검진+교정) | 다양 | 아이+부모 유입 = 300만원+ | 20만원 |

**핵심:** CPA 10만원이어도 임플란트 LTV 200만원이면 ROI 20배. CPA만 줄이려 하면 오히려 성장 기회를 놓친다.

**가족 단위 LTV:** 동백은 가족 중심 지역. 1명 유입 → 가족 3~4명 전환 가능. 소아 환자 유치 → 부모 임플란트/교정으로 이어지는 승수 효과.

→ 진입 시술(스케일링, 충치)로 유입 → 신뢰 구축 → 고단가 시술(임플란트, 교정) 전환이 핵심 퍼널.

| 일간 | 플레이스 순위 | "동백 치과" 키워드 순위 변동 (자동 추적) |
| 주간 | 로덴, 센트럴, 탑치과 | SA 광고 소재 캡처, 신규 블로그 포스팅 |
| 월간 | 상위 5개 치과 | 리뷰 수 변화, 가격 변동, 플레이스 정보 변경 |

**대응 시나리오:**
- 경쟁사 가격 인하 → 가격 비교 콘텐츠 업데이트
- 경쟁사 SA 시작 → 입찰 전략 재조정
- 경쟁사 리뷰 급증 → 리뷰 확보 캠페인 강화

### 대시보드

- **도구:** 구글 스프레드시트 (무료, 충분한 기능)
- **실시간 뷰:** 오늘 광고비 소진, 클릭수, 전화수
- **주간 뷰:** 채널별 CPA/CPL 추이, 키워드 Top10/Bottom10
- **월간 뷰:** ROAS, LTV 기반 수익성, 전월 대비 변화
- **원장 1페이지 리포트:** 숫자 3개 — 이번 주 전화 수, CPA, 전월 대비 증감
- **알림:** CPA 급등, 예산 소진률 이상, 품질지수 하락 시 카톡 알림

## Part 12. 운영 인력

### 필요 인력

블로그 주 1~2회 + 리뷰 관리 + SA 모니터링 + 플레이스 소식을 합하면 하루 1~2시간의 마케팅 업무가 발생한다.

| 옵션 | 비용 | 적합 상황 |
|------|------|----------|
| 원장 직접 + 데스크 직원 분담 | 0원 | 저예산, 초기 |
| 마케팅 파트타임 1명 | 월 80~120만원 | 중예산 |
| 마케팅 대행 + 원장 감수 | 월 100~200만원 | 고예산, 시간 없을 때 |

### 업무 분배 (저예산 기준)

| 담당 | 업무 | 주당 시간 |
|------|------|----------|
| 원장 | 블로그 팩트 음성 녹음, 리뷰 답변 감수 | 1시간 |
| 데스크 | 리뷰 요청, 플레이스 소식, 사진 업로드 | 2시간 |
| AI/자동화 | 블로그 초안, SA 데이터 수집, 리포트 생성 | - |
| 외부 | 심의 신청, 전문 촬영 (일회성) | - |

## Part 13. 기존 광고업체 전략 분석

### 13.1 업체들의 공통 전략 (복제 가능)

1. 고의도 키워드 집중 집행 (지역+시술+가격/후기)
2. 제외키워드 주 1~2회 갱신
3. 시술별 랜딩페이지 분리
4. 광고문안 A/B 테스트 후 승자 확장
5. 플레이스/리뷰 운영 병행
6. 전화·예약 전환 KPI 중심 보고

### 13.2 기존 업체의 약점

- 수작업 비중이 높고 수정 주기가 느림 (주간 최적화)
- 운영 논리가 블랙박스 ("최적화했습니다" 보고)
- 광고/플레이스/블로그를 분절적으로 관리
- 월 200~500만원 패키지 비용

### 13.3 우리의 차별화 포인트

| 항목 | 기존 업체 | 우리 |
|------|----------|------|
| 최적화 주기 | 주간 | 일간 감시+즉시 알림 |
| 채널 관리 | 분절 (광고/플레이스/블로그 따로) | 5채널 통합 (동일 키워드 체계) |
| 의사결정 | "최적화했습니다" | 근거 로그 + 의사결정 트리거 |
| 의료 리스크 | 사후 대응 | 과장/단정 표현 사전 경고 |
| 환자 관점 | 신환 유입만 | LTV 기반 (유입+리텐션+가족 확장) |
