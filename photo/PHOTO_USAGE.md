# 실제 병원 사진 사용 가이드

업데이트: 2026-03-12

## 원본 위치
- `photo/*.jpeg`

## 웹사이트 사용 위치
- 외관/간판 사진: 메인 히어로, 오시는길 섹션
  - 예: `KakaoTalk_Photo_2026-03-12-21-57-37 001.jpeg`
- 접수/데스크 사진: 병원 소개, 신뢰 요소 섹션
  - 예: `KakaoTalk_Photo_2026-03-12-21-57-47 013.jpeg`
- 진료실/장비 사진: 시설 소개, 진료 환경 섹션
  - 예: `KakaoTalk_Photo_2026-03-12-21-57-46 007.jpeg`
- 내부 동선/복도 사진: 시설 소개 갤러리
  - 예: `KakaoTalk_Photo_2026-03-12-21-57-45 001.jpeg`

## 프론트엔드 경로
- 복사본: `site/public/images/clinic/*.jpeg`
- Next.js에서 사용 예시:
  - `/images/clinic/KakaoTalk_Photo_2026-03-12-21-57-37 001.jpeg`

## 체크리스트
- 환자 얼굴/개인정보 노출 여부 최종 확인
- 사진 EXIF 위치정보 제거 여부 확인(배포 전)
- 모바일 최적화를 위해 webp 변환본 추가 권장
