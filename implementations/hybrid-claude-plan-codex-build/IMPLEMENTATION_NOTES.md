## Claude PLAN 요약

- 디자인 톤은 `심플 + 전문 + 친근`을 유지하고, 네이비와 민트 포인트 컬러를 중심으로 구성한다.
- 정보 구조는 홈, 병원소개, 진료과목, 의료진, 시설/장비, 오시는길, 상담/예약의 7개 축으로 유지한다.

- 기술 스택은 Next.js App Router + TypeScript + Tailwind CSS 4를 기본으로 한다.

## 구현 반영 메모

- App Router 기준으로 `app/` 아래 최소 7개 페이지를 구성했다.
- 공통 데이터는 `lib/site-data.ts`에 분리해 페이지별 카피와 운영 정보를 반복 없이 재사용한다.
- 톤은 과한 마케팅 문구보다 신뢰와 접근성을 우선하는 한국 치과 웹 스타일로 정리했다.
- 실사진은 `public/images/clinic/`로 복사해 `next/image`로 렌더링한다.
- 의료진 상세 정보가 비어 있는 점은 허위 정보 대신 진료 철학 중심 콘텐츠로 처리했다.
- 위치 페이지는 실제 지도 연동 전 단계라 임베드 자리 표시 영역을 남겼다.
- 문의 폼은 정적 UI만 제공하며 백엔드 연동은 포함하지 않았다.
